# 💊 Pharmacy Management System

## 🚀 Overview

The Pharmacy Management System is a desktop-based application built using **C# (.NET Framework)** and **SQL** to streamline pharmacy operations. It provides efficient management of inventory, sales, purchases, and user access with a secure and structured workflow.

This project was developed as part of a Database Systems course.

---

## 🛠️ Tech Stack

* **Frontend:** C# (Windows Forms)
* **Backend:** .NET Framework
* **Database:** SQL Server
* **IDE:** Visual Studio

---

## ✨ Features

### 🔐 Authentication & Authorization

* Secure login system
* Role-based access control
* Main Admin approval system for new admin accounts

### 📦 Inventory Management

* Add, update, delete products
* Track stock levels, pricing, and categories
* Organized storage using rack/row indexing

### 🏭 Manufacturer Management

* Manage manufacturer details
* Full CRUD operations

### 🛒 Purchase & Sales Module

* Record product purchases
* Handle product sales
* Maintain transaction logs

### 📊 Reporting & Insights

* Sales reports
* Expense tracking
* Inventory insights for decision-making
